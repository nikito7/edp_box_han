# Testados

**Alimentação directa na HAN**

Monofásicos:

- ```Kaifa MA109P```

- ```Landis+Gyr```

- ```Sagemcom CX1000```

- ```Sagemcom S212```

Trifásicos:

- ```Janz B2801 GPRS```

- ```Sagemcom CX2000```

Notas:

Landis+Gyr: Serial mN1/mN2.
